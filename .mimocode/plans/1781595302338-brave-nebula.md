# 主题更换功能实现计划

## 背景

当前 webview 完全依赖 VSCode 自动注入的 `--vscode-*` CSS 变量实现主题适配，没有主题切换功能。

用户希望实现：通过设置选择主题颜色，默认跟随 VSCode 主题。

## 核心技术方案

### 获取已安装主题列表

```typescript
// 通过 vscode.extensions.all 遍历所有扩展
const themes = vscode.extensions.all
  .flatMap(ext => {
    const contributes = ext.packageJSON?.contributes;
    if (!contributes?.themes) return [];
    return contributes.themes.map(theme => ({
      id: theme.id,
      label: theme.label,
      uiTheme: theme.uiTheme, // vs-dark, vs, hc-black, hc-light
      path: path.join(ext.extensionPath, theme.path),
      extensionId: ext.id
    }));
  });
```

### 主题切换实现

1. **Extension 侧**：读取主题 JSON 文件，提取颜色定义，发送给 webview
2. **Webview 侧**：接收颜色映射，通过 `document.documentElement.style.setProperty()` 覆盖 CSS 变量

## 实现步骤

### Phase 1: 验证可行性（最小案例）

在 `MarkdownEditorProvider.ts` 中添加命令，打印获取到的主题列表，验证能否正确获取。

**文件**: `src/MarkdownEditorProvider.ts`

### Phase 2: 完整实现

#### 2.1 添加配置项

**文件**: `package.json`

```json
{
  "markdownWysiwyg.colorTheme": {
    "type": "string",
    "default": "auto",
    "description": "%markdownWysiwyg.colorTheme.description%"
  }
}
```

配置值说明：
- `"auto"` — 跟随 VSCode 主题（默认）
- `"<主题ID>"` — 指定主题 ID（如 `"GitHub Dark"`、`"Dracula"`）

#### 2.1.1 添加命令

**文件**: `package.json`

```json
{
  "command": "markdownWysiwyg.selectTheme",
  "title": "%markdownWysiwyg.selectTheme.title%"
}
```

用户可通过命令面板选择主题，同时支持在 settings.json 中直接填写主题 ID。

#### 2.2 Extension 侧逻辑

**文件**: `src/themeManager.ts`（新建）

- `getAllThemes()`: 获取所有已安装主题列表
- `getThemeColors(themePath)`: 读取主题 JSON，提取颜色映射
- `applyTheme(panel, themeId)`: 向 webview 发送主题颜色

**文件**: `src/MarkdownEditorProvider.ts`

- 监听配置变化 `onDidChangeConfiguration`
- 在 `resolveCustomEditor` 中应用当前主题
- 监听 `vscode.window.onDidChangeActiveColorTheme`（auto 模式）

#### 2.3 消息类型扩展

**文件**: `shared/messages.ts`

```typescript
// Extension → WebView
| { type: "setTheme"; colors: Record<string, string> }
```

#### 2.4 Webview 侧逻辑

**文件**: `webview/index.ts`

- 监听 `setTheme` 消息
- 遍历 colors，调用 `document.documentElement.style.setProperty(key, value)`

#### 2.5 CSS 变量映射

需要建立 VSCode 主题颜色 ID 到 webview CSS 变量的映射表：

| VSCode 主题颜色 ID | Webview CSS 变量 |
|---|---|
| `editor.background` | `--vscode-editor-background` |
| `editor.foreground` | `--vscode-editor-foreground` |
| `textLink.foreground` | `--vscode-textLink-foreground` |
| ... | ... |
